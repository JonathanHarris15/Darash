# Release Notes - v0.1.3

## Key Changes
- **Release Notes Feature:** Introducing a new in-app viewer for version updates, ensuring you never miss a new feature or improvement.
- **Styling Playground:** Added a developer-focused playground to iterate on theme tokens and UI components in real-time.
- **Spellcheck:** Spellcheck has been added to the outlines and notes editors
- **Architectural Cleanup:** Major refactoring for better stability, including dedicated scene managers and improved layout logic.
- **Improved Translation Support:** Enhanced multi-translation chapter loading and interlinear display stability.
