# Release Notes - v0.1.4

## ✨ Architectural Refinement
- **Major Refactoring**: To comply with architectural mandates, four core files were split into specialized sub-modules, bringing all primary facades under the 300-line soft limit.
- **Improved Maintainability**: Logic for dock management, scene state, study input handling, and tree population has been isolated into focused managers.

## 🐛 Bug Fixes
- **UI Connectivity**: Resolved issues where new outlines and notes were not automatically opening or connecting to the main interface.
- **Facade Restoration**: Restored several critical geometric and navigation methods in the `ReaderScene` that were lost during the initial refactor.
- **Note Editor Fixes**: Corrected an initialization bug that prevented some notes from being opened.
- **Syntax Correction**: Fixed a script error in the `StudyTreeWidget` logic.

## 🚀 Stability
- Passed all 10 core UI tests and verified domain integrity with the architectural health check tool.
