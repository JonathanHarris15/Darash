# Release Notes - v0.1.5

## ✨ UI Enhancements
- **Rich Text Headers**: Improved header styling in the notes editor. Headers now correctly apply and sync font size (H1: 22pt, H2: 18pt, H3: 14pt) and bold weight.
- **Formatting Toolbar**: Enhanced the formatting toolbar to accurately reflect the current block's heading level and styling.

## 🐛 Bug Fixes & Stability
- **Initialization Safety**: Added safety checks to the formatting toolbar to prevent crashes during editor startup.
- **Test Infrastructure**: Resolved multiple integration test failures related to `QGraphicsItem` mock mismatches and layout timing.
- **Improved Reliability**: Fixed issues with scene overlay rendering (arrows and marks) by ensuring coordinate maps are populated before rendering.

## 🚀 Quality Assurance
- **New Tests**: Added comprehensive component tests for `FormattingToolBar` heading sync and application logic.
- **Green Suite**: Verified the stability of the entire application with 123 passing tests (including E2E, integration, and unit tests).
