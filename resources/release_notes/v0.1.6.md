# Release Notes - v0.1.6

## Key Changes
- **Hierarchical Numbered Lists:** Added tiered numbering styles for nested lists in the Note Editor. Indenting a list item now automatically cycles its style:
    1. Level 1: Decimal (1, 2, 3...)
    2. Level 2: Lower Alpha (a, b, c...)
    3. Level 3: Lower Roman (i, ii, iii...)
- **Improved Dock Stability:** Added robust error handling to high-frequency UI timers to prevent crashes when closing panels.
- **Shutdown Protection:** Background timers are now explicitly stopped during application shutdown to ensure a clean exit.
- **Stacked Tab Fix:** Fixed a bug where close buttons on tabified docks were unresponsive.
- **Font Rendering Fixes:** Added guards to prevent negative font sizes, resolving startup warnings and potential rendering issues.

## Coming Soon
- **Improved Outline Navigation:** Faster jumping between deeply nested outline sections.
- **Advanced Search Filters:** Scoping searches by specific study marks or symbol types.
