Create a spellcheck feature that can be integrated into the Notes Editor and Outline Editor. Eventually it will be used in other places so it should be designed to be flexible and reusable.

Do not worry about improper grammar or sentence structure. Just make a spellcheck that can be used to check for misspelled words. But leave the design open for future expansion.

The spellcheck should put a red bar under the misspelled word. The user can then right click on the misspelled word to see a list of suggested corrections. The user can then click on a suggestion to replace the misspelled word with the suggested correction. The user should also be able to add it to a dictionary of ignored words.

The styling of the list of suggested corrections should be consistent with everything that has been implimented and should refer to the styling sheet.
